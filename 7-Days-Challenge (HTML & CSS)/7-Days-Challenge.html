(Hyper Text Markup Language) Html is the code that is used to structure a web page and its content.

HTML Basic Structure

  1. <!DOCTYPE html>: Declares the document type and HTML version.
  2. <html>: The root element of an HTML page.
  3. <head>: Contains meta-information about the document (e.g., title, links to CSS).
  4. <title>: Sets the title of the document, shown in the browser tab.
  5. <meta>: Provides metadata like character set, author, and viewport settings.
  6. <link>: Links to external resources like stylesheets.
  7. <style>: Contains internal CSS styles.
  8. <body>: Contains the content of the document (text, images, etc.).
The component used to design the structure of websites are called HTML tags (a container for some content or other HTML tags.)

Here is a list of some important HTML tags

Text Formatting Tags

  1. <h1> - <h6>: Headings, <h1> is the highest level and <h6> is the lowest.
  2. <p>: Paragraph of text.
  3. <br>: Line break.
  4. <hr>: Horizontal rule (line).
  5. <strong>: Important text (typically bold).
  6. <em>: Emphasized text (typically italic).
  7. <ul>: Unordered list.
  8. <ol>: Ordered list.
  9. <li>: List item.
  10. <dl>: Description list.
  11. <dt>: Term in a description list.
  12. <dd>: Description of the term.

Links and Images

  1. <a>: Anchor tag for hyperlinks.
  2. <img>: Embeds images.
  3. <map>: Defines a client-side image map.
  4. <area>: Defines clickable areas in an image map.

Multimedia

  1. <audio>: Embeds audio content.
  2. <video>: Embeds video content.
  3. <source>: Specifies multiple media resources for <audio> or <video>.
  4. <track>: Provides text tracks for <video> (like subtitles).

Tables

  1. <table>: Defines a table.
  2. <tr>: Table row.
  3. <th>: Table header cell.
  4. <td>: Table data cell.
  5. <caption>: Table caption.
  6. <thead>: Group header content in a table.
  7. <tbody>: Group body content in a table.
  8. <tfoot>: Group footer content in a table.

Forms

  1. <form>: Defines an HTML form for user input.
  2. <input>: Input field for forms.
  3. <textarea>: Multi-line text input.
  4. <button>: Button for form submission or other actions.
  5. <select>: Dropdown list.
  6. <option>: Options in a dropdown.
  7. <label>: Defines a label for an input element.
  8. <fieldset>: Groups related elements in a form.
  9. <legend>: Caption for a <fieldset>.

Semantic HTML

  1. <header>: Defines a header for a document or section.
  2. <nav>: Defines navigation links.
  3. <section>: Defines a section in a document.
  4. <article>: Independent content piece.
  5. <aside>: Content aside from the main content (sidebar).
  6. <footer>: Footer for a document or section.
  7. <main>: Main content of a document.

Scripting

  1. <script>: Embeds JavaScript.
  2. <noscript>: Defines alternative content for users without scripts enabled.

Other Important Tags

  1. <div>: A container for other elements (block-level).
  2. <span>: A container for inline elements.
  3. <iframe>: Embeds another HTML page within the current page.


HTML Document Structure

1. Document Structure

  • Start with <!DOCTYPE html> to set the document type.
  • Use <html> to contain all your content.
  • Put metadata in <head> and visible content in <body>.

2. Element Nesting

  • Always close your tags (e.g., <div></div>).
  • Make sure elements are properly nested; you can't put a <p> tag inside a <div> and then another <div> inside that <p>.

3. Attributes

  • Use attributes for extra information about elements (like href in <a> tags).
  • Put attribute values in double quotes (" "), but single quotes (' ') are okay too.
  • Always use lowercase for element names and attributes to keep things consistent.

4. Semantic HTML

  • Use semantic HTML elements like <header>, <footer>, <article>, <section>, and <nav> to help with accessibility and search engines.
  • These elements help explain the content to browsers and assistive technologies.

5. Accessibility

  • Add alt attributes for images to describe them for screen readers.
  • Make links clear and meaningful (e.g., use <a href="contact.html">Contact Us</a> instead of <a href="contact.html">Click here</a>).
  • Use heading tags (<h1> to <h6>) in order to create a clear structure.

6. Character Encoding

  • Declare character encoding in the <head> section with <meta charset="UTF-8"> to display characters correctly.

7. Comments

  • Use comments (<!-- Comment here -->) to leave notes in your code that won't show up on the webpage.

8. Validation

  • Check your HTML for errors with tools like the W3C HTML Validator to make sure it follows standards.

9. Avoid Old Tags

  • Don't use outdated tags (like <font>, <center>, and <marquee>). Use CSS for styling instead.

10. Whitespace Management

  • HTML ignores extra spaces, but using consistent formatting helps readability. Indent nested elements for clarity.

11. External Resources

  • Link to external stylesheets and scripts using <link> and <script> tags. Put scripts at the end of the <body> for better loading speed.

12. Responsive Design

  • Use the viewport tag (<meta name="viewport" content="width=device-width, initial-scale=1.0">) for responsive design on mobile devices.

Section 1 - Basic Selectors

  1. * - Universal Selector
  2. p - Type Selector
  3. .classname - Class Selector
  4. #elementId - ID Selector
  5. input[type="text"] - Attribute Selector

Combinator Selectors

  1. div p - Descendant Selector (space)
  2. ul > li - Child Selector
  3. h1 + p - Adjacent Sibling Selector
  4. h1 ~ p - General Sibling Selector

Grouping Selector

  1. h1, h2, h3 - Group Selector

Pseudo-Class Selectors

  1. a:hover - Hover Pseudo-Class
  2. input:focus - Focus Pseudo-Class
  3. p:first-child - First-Child Pseudo-Class
  4. p:last-child - Last-Child Pseudo-Class
  5. li:nth-child(2) - Nth-Child Pseudo-Class
  6. div:not(.special) - Negation Pseudo-Class

Pseudo-Element Selectors

  1. h1::before - Before Pseudo-Element
  2. p::after - After Pseudo-Element
  3. p::first-line - First-Line Pseudo-Element
  4. p::first-letter - First-Letter Pseudo-Element

Attribute Selectors (Detailed)

  1. [disabled] - Attribute Selector
  2. [type="submit"] - Exact Attribute Selector
  3. a[href^="https"] - Attribute Starts With Selector
  4. img[src$=".png"] - Attribute Ends With Selector
  5. a[href*="example"] - Attribute Contains Selector
